📈 Amazon Stock Analysis Dashboard
📌 Project Overview

The Amazon Stock Analysis Dashboard is an interactive Microsoft Excel dashboard designed to analyze Amazon's historical stock market performance. It provides insights into stock price movements, trading volume, daily trends, monthly averages, and yearly performance through dynamic charts, KPIs, and slicers.

🎯 Objective

The primary objective of this dashboard is to:

Monitor Amazon stock price performance over time.
Compare Open, High, Low, and Close prices.
Analyze trading volume trends.
Identify monthly and yearly stock price patterns.
Enable users to explore stock data interactively using filters.
📊 Dashboard Features
Interactive Day, Month, and Year Slicers
Dynamic KPI Cards for Average Open, High, Low, and Close Prices
Open vs High Price Comparison by Day
High vs Low Price Analysis
Trading Volume Analysis
Monthly Average Adjusted Close Price
Yearly Adjusted Close Price Trend
Day-wise Value Distribution
Fully interactive visualizations that update based on selected filters
📈 Key Performance Indicators (KPIs)
📌 Average Open Price
📌 Average High Price
📌 Average Low Price
📌 Average Close Price
📊 Trading Volume
📅 Monthly Adjusted Close Price
📈 Yearly Stock Performance
